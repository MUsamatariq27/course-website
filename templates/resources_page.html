Useful resources for CSCB20

W3Schools

W3Schools is an educational website for learning web technologies online. Content includes tutorials and references relating to HTML, CSS, JavaScript, Python, SQL and Bootstrap

Piazza

Piazza is a discussion forum with many features that are designed specifically for use with courses. We encourage you to post questions (and answers!) on piazza, and read what other questions your classmates have posted. The CSC B20 Piazza page will be monitored by your instructors and teaching assistants. However, since piazza is run by another company, we also encourage you to read the privacy policy carefully and only sign up if you are comfortable with it. Contact the course coordinator if you have additional questions about using piazza.

Python Documentation

There is extensive Python documentation; you can use it online or click the first link to download the documentation to your own computer. (Please do not download it to your Teaching Labs account; use the online version instead.) Reading this documentation is not mandatory for the course, and any part of it which is not covered in the lectures would not be considered part of the course. But it is a good idea, especially towards the end of the course, to begin to learn to use it. Please, ask if you have any questions!
